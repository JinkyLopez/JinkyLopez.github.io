Introduction

HTML (Hypertext Markup Language) and CSS (Cascading Style Sheets) are the fundamental technologies for building web pages. HTML provides the structure of the page, while CSS styles the elements within it.

Here's a brief history of HTML and CSS:

HTML

  • 1989-1990: HTML was invented by Sir Tim Berners-Lee while working at CERN. The first version, HTML 1.0, was very basic and focused on structuring documents with simple tags like [head], [title], [body], and [h1] to [h6] for headings ("[]" should be replaced with "<>").
  • 1993-1994: HTML 2.0 was published as an Internet Engineering Task Force (IETF) RFC (Request for Comments) in 1995, introducing features like forms and tables.
  • 1995-1997: HTML 3.2 was released, adding support for features like image maps and text flow around images.
  • 1997-1999: HTML 4.0 and HTML 4.01 were released, bringing improved support for scripting and styling through the introduction of [script] and [style] elements.
  • 2000-2014: HTML underwent a period of stagnation, with no major updates until HTML5.
  • 2014: HTML5 was published as a W3C Recommendation, introducing many new features like semantic elements [header], [footer], [nav], etc., multimedia elements [video], [audio], and the Canvas API for drawing graphics.

CSS
  • 1994-1996: CSS was proposed by HÃ¥kon Wium Lie and Bert Bos at CERN, aiming to separate the presentation from the content of web pages.
  • 1996-1997: CSS1 was introduced as a W3C Recommendation in December 1996, providing basic styling capabilities like fonts, colors, and spacing.
  • 1998-1999: CSS2 was published as a W3C Recommendation in May 1998, adding support for positioning, media types, and more advanced styling features.
  • 2000-2010: CSS underwent various revisions and updates, but CSS2 remained the standard for many years.
  • 2011-2012: CSS3 modules started to be developed independently, allowing for more granular control over styling elements like borders, backgrounds, and animations.
  • 2014-present: CSS3 is still evolving, with new modules being added to extend the capabilities of CSS, including CSS Grid Layout and CSS Flexbox for layout, as well as CSS Variables and Custom Properties for dynamic styling.

HTML Basics

HTML is the standard markup language for creating web pages. It defines the structure and content of a web page using a variety of tags and attributes.

In HTML, you use tags to define different elements such as headings, paragraphs, links, images, and more. Here's a simple example: 

                    <h1>Hello, world!</h1>
                    <p>This is a paragraph.</p>
                    <div>This is a separator or section for your codes.</div>

You can check the other tags in the reference below.

Media Elements in HTML

Global variables are in fact properties of the global object. In web pages the global object is window, so you can set and access global variables using the window.variable syntax. 

    1. Images (img):
        The <img> element is used to embed images in a webpage.
        Example:
        <img src="link to image" >
    
    2. Links (a):
        The <a> element is used to create hyperlinks to other webpages or resources.
        Example:
        <a href="https://example.com"> Name to the link </a>
    
    3. Videos (video):
        The <video> element is used to embed videos in a webpage.
        Example:
        <video width="320" height="240" controls>
        <source src="video.mp4" type="video/mp4">
            Your browser does not support the video tag.
        </video>
    
    4. Audio (audio):
        The <audio> element is used to embed audio files in a webpage.
        Example:
        <audio controls>
        <source src="audio.mp3" type="audio/mpeg">
            Your browser does not support the audio tag.
        </audio>
CSS Basics

CSS is a stylesheet language that controls the presentation of HTML elements on a web page. It allows you to style the layout, colors, fonts, and other visual aspects of your web page.

CSS works by selecting HTML elements and applying styles to them. For example: 

    h1 {
        color: blue;
        font-size: 24px;
    }

    p {
        line-height: 1.5:
        text-align: center;
    }

There are many other ways to "style" your HTML. If you'd like more, check the reference below.

Selectors in CSS

Selectors are patterns used to select the elements you want to style in CSS. There are various types of selectors, including element selectors, class selectors, ID selectors, and more. 

        /* Selecting elements by tag name */
        p {
            color: blue;
        }

        /* Selecting elements by class */
        .intro {
            font-size: 18px;
        }

        /* Selecting elements by ID */
        #main-title {
            text-align: center;
        }

        /* Selecting elements by attribute */
        input[type="text"] {
            border: 1px solid #ccc;
        }

        /* Selecting nested elements */
        .container li {
            list-style-type: circle;
        }

        /* Selecting elements based on their relationship */
        .header + p {
            margin-top: 20px;
        }
Box Model in CSS

The box model is a fundamental concept in CSS that defines how elements are rendered on a web page. It consists of margin, border, padding, and content areas around each HTML element. Below are some examples: 

    1. Content: Actual content of the element.
    .box {
        width: 200px;
        height: 100px;
    }
    
    2. Padding: Space between content and border.
    .box {
        padding: 20px;
    }
    
    3. Border: Area surrounding padding and content.
    .box {
        border: 2px solid black;
    }
    
    4. Margin: Space outside the border.
    .box {
        margin: 20px;
    }
Flexbox in CSS

Flexbox is a layout model in CSS that allows you to create flexible and efficient layouts. It simplifies the process of aligning and distributing space among items in a container, even when their size is unknown or dynamic. 

    1. Flex Container Properties:
        a. display: Defines a flex container.
            Example:
            .container {
                display: flex;
            }
    
        b. flex-direction: Specifies the direction of flex items.
            Example:
            .container {
                flex-direction: row;
            }
    
    2. Flex Item Properties:
        a. flex-grow: Specifies how much a flex item will grow.
            Example:
            .item {
                flex-grow: 1;
            }
    
        b. flex-shrink: Specifies how much a flex item will shrink.
            Example:
            .item {
                flex-shrink: 0;
            }
    
    3. Alignment Properties:
        a. justify-content: Aligns flex items along the main axis.
            Example:
            .container {
                justify-content: center;
            }
    
        b. align-items: Aligns flex items along the cross axis.
            Example:
            .container {
                align-items: center;
            }
